# Formulários

- Formulários:
  - São campos que seram criados para o usuário interagir ou digitar alguma mensagem ou escolher alguma opção que seram enviadas para algum lugar.

```html
  <form></form>
```

  > Usada para criar o formulário.

```html
  <input />
```

```html
  <textarea></textarea>
```

```html
  <select></select>
```
